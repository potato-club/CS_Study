# Key

> 검색, 정렬시 Tuple을 구분할 수 있는 기준이 되는 Attribute.

### 1. Candidate Key (후보키)

```ABAP
> Tuple을 유일하게 식별하기 위해 사용하는 속성들의 부분 집합
> (기본키로 사용할 수 있는 속성들)

아래 2가지 조건을 만족해야 한다.
1. 유일성: key로 하나의 Tuple을 유일하게 식별할 수 있음
2. 최소성: 꼭 필요한 속성으로만 구성
```

### 2. Primary Key (기본키)

```ABAP
> 후보키 중 선택한 Main Key

특징
1. Null 값 ❌
2. 값 중복 ❌
```

### 3. Alternate Key (대체키)

> 후보키 중 기본키를 제외한 나머지 키 = 보조키

### 4. Super Key (슈퍼키)

> 유일성 ⭕️, 최소성 ❌

### 5. Foreign Key (외래키)

> 다른 릴레이션의 기본키를 그대로 참조하는 속성의 집합
